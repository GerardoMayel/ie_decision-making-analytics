# Analytics for Decision-Making Portfolio

This repository contains my portfolio of coding exercises, case studies, and practical assignments completed during the **Decision-Making Analytics [STELLANTIS-DMA-26-C2]** program at IE University.

The coursework focuses on using Python and Pandas to analyze data distributions, implement the scientific method, and explore how data quality and cognitive biases impact corporate decision-making.

---

## 🛠️ Data Science Tech Stack
* **Language:** Python
* **Libraries:** `pandas`, `numpy`, `matplotlib`, `seaborn`
* **Environments:** Google Colab, VS Code

---

## 📁 Repository Structure & Course Modules

### 🔹 Module 1: Introduction to Decision-Making Analytics
* **Core Concepts:** Defining the analytics spectrum (Descriptive, Diagnostic, Predictive, Prescriptive) and studying cognitive biases and fallacies.
* **Case Studies & Readings:**
  * **The Elizabeth Holmes Case:** Examining decision-making with no data.
  * **The Enron Case:** Analyzing bad data and corporate outcomes.
  * **The Itiel Dror Paper:** Reviewing "Six Fallacies and the Eight Sources of Bias" in expert decision-making.
  * **Bad Data in Academic Research:** Reviewing *The Lancet* 1998 MMR vaccine paper retraction and its public health consequences.
* **Activities:**
  * `W1 Discussion 1:` Data Analysis in a Day-to-Day Context.
  * `W1 Discussion 2:` Dealing with Cognitive Biases.
  * `W1 Discussion 3:` Consequences of Bad Data in Academic Research.
  * `Practice Activity:` Country GDP Selection (Manipulating Data with Python in Google Colab).

### 🔹 Module 2: Questioning Graphs — Handling and Interpreting Data
* **Core Concepts:** Hypothesis formulation, data gathering, drawing conclusions, data preparation, quality control, simple linear regression, and correlation vs. causation.
* **Activities & Assignments:**
  * `W2 Discussion 1:` Discussing Correlations.
  * `Practice Activities:` Hypothesis Testing, Data Analysis, Linear Regression, and Correlation Analysis.
  * `Assignment A2.1:` Data Analysis and Graph Creation.

### 🔹 Module 3: Thriving with Data
* **Core Concepts:** Understanding what makes good data analysis, managing the quality vs. quantity dilemma, and reviewing issues in Generative AI.
* **Case Studies:**
  * **The Subprime Crisis:** Analyzing same data, different outcomes.
  * **The UPS Case:** Reviewing good data and decision-making in logistics.
* **Activities & Assignments:**
  * `W3 Discussion 1:` Data and Their Impact on Decisions.
  * `W3 Discussion 2:` Strengths and Limitations of Pandemic Management through Computer Science.
  * `W3 Discussion 3:` Bad Data Quality and Wrong Decisions.
  * `W3 Discussion 4:` Issues in Generative AI.
  * `W3 Discussion 5:` Key Takeaways in Decision-Making Analytics.
  * `Assignment A3.1 (Final Project):` Decision-Making with the Scientific Method.
